# Zenith Mindspace

## Architecture for the Mind

Zenith Mindspace is a comprehensive mental health and personal development platform offering counseling, performance coaching, workshops, and educational programs designed to support psychological health and personal evolution.

## Website Features

- **Responsive Design**: Fully compatible with mobile, tablet, and desktop devices
- **Modern UI**: Built with Tailwind CSS for a clean, professional appearance
- **Interactive Elements**: 
  - Collapsible FAQ section
  - Mobile-friendly navigation menu
  - Smooth scrolling between sections
  - Contact form with validation

## Sections

1. **Home**: Hero section with call-to-action
2. **About**: Mission statement and core values
3. **Services**: Three main service offerings
4. **Facilitators**: Team member profiles
5. **Community**: Call to join the community
6. **Why Choose Us**: Key differentiators
7. **FAQ**: Common questions and answers
8. **Contact**: Contact form and information

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for CDN resources (Tailwind CSS, Font Awesome, Google Fonts)

### Running the Website

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Alternatively, serve the files using a local web server:

```bash
# Using Python (if available)
python -m http.server 8000

# Or using Node.js (if available)
npx serve .

# Then open http://localhost:8000 in your browser
```

## Customization

### Images

The website uses placeholder images from Picsum. To replace with actual team photos:

1. Save team member photos to `images/team/` directory
2. Update the `src` attributes in `index.html`:
   - `images/team/safna-ershad.jpg` for Safna Ershad
   - `images/team/shabas-shamnad.jpg` for Shabas Shamnad
   - `images/team/anil-kumar.jpg` for Anil Kumar V.S
   - `images/team/ambika-kumari.jpg` for Ambika Kumari

### Styling

- Custom styles are defined in the `<style>` tag within `index.html`
- Colors use Tailwind's teal palette as primary (#0f766e)
- Fonts: Inter for body text, Playfair Display for headings

### Content

Edit the HTML directly in `index.html` to update text, links, or structure.
